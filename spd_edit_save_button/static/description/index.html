Edit Save and New Button In Odoo17

Overview

Using this app you can edit, and save records. You can also create new records and discard any changes as needed.

Features

Community & Enterprise Support.
Set the form readonly when rendering the form.
Available in Odoo 17.0 Community and Enterprise.
When click the edit button form will change to edit mode.
When click the new button then create a new record.
You can discard the changes.
